# 怎么加照片

## 步骤

**第一步：把照片放进这个文件夹**

命名规则：`YYYY-MM-DD-01.jpg`（日期 + 序号）

例如：
- `2024-09-15-01.jpg`
- `2024-09-15-02.jpg`（同一天多张）
- `2023-12-25-01.jpg`

**第二步：打开 `src/data/photos.json`，加一条记录**

```json
{ "src": "/photos/2024-09-15-01.jpg", "date": "2024-09-15", "width": 4032, "height": 3024 }
```

字段说明：
- `src`：文件路径，从 `/photos/` 开始
- `date`：拍摄日期，格式 `YYYY-MM-DD`（照片会按这个日期排序）
- `width` / `height`：照片原始尺寸（像素）
- `title`：可选，大多数照片不需要

**第三步：git push → Vercel 自动部署**

---

## 宽高怎么查

Mac：右键照片 → 显示简介 → 尺寸

---

## 注意事项

- 照片自动按日期从新到旧排列
- `date` 只需要年月日，同月第一张会显示月份标记
- 大多数照片不写 `title`，只有特别值得留一句话的才写
- 文件名建议用 `.jpg` 或 `.webp`，`.heic` 不支持
